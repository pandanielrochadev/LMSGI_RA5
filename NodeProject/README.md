# RA5 - Conversió de documents

Aplicació web desenvolupada amb Node.js i Express per convertir dades entre formats.

## Funcionalitats

- Conversió de JSON a XML
- Funció de conversió de XML a JSON implementada

## Tecnologies utilitzades

- Node.js
- Express
- HTML
- JavaScript

## Execució

Instal·lar dependències:

```bash
npm install
```

Executar el servidor:

```bash
node server.js
```

Obrir al navegador:

```text
http://localhost:3000
```
