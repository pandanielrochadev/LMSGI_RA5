# RA5 - Activitat 3

Aplicació web desenvolupada amb Node.js, Express i PokeAPI.

## Funcionalitats

- Cerca un Pokémon introduint el seu nom.
- Obté la informació des de la PokeAPI.
- Converteix el JSON rebut a format XML.
- Mostra la imatge (sprite) del Pokémon.
- Mostra les habilitats del Pokémon.

## Tecnologies utilitzades

- Node.js
- Express
- xml-js
- PokeAPI
- HTML
- JavaScript

## Instal·lació

```bash
npm install
```

## Execució

```bash
node server.js
```

## Navegador

```text
http://localhost:3000
```
